# Labyrinth Snippets

This is a local Zed snippet extension containing the C# snippets ported from
`.vscode/chickensoft.code-snippets` and `.vscode/godot.code-snippets`.

Install it with `zed: install dev extension` and select this directory:

```text
.zed/labyrinth-snippets
```

Zed currently uses only the first entry when a snippet has multiple prefixes,
so VS Code prefix aliases were expanded into separate snippets.
